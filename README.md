# My Assignment 01 by iCodeGuru

## 📌 Project Overview
This project is a **responsive website** built using **HTML and CSS**. The layout is designed to be fully adaptable across different screen sizes, including **large screens, tablets, and mobile devices**. The project includes a **navigation bar, hero section, posts, featured work, and a footer**.

## 🛠️ Technologies Used
- **HTML5** - For structuring content  
- **CSS3** - For styling and responsiveness  
- **Media Queries** - To make the design responsive  

## 📌 Features
- ✅ **Fully Responsive Design** – Works on desktops, tablets, and mobile screens  
- ✅ **Navigation Bar** – Simple, modern navigation for easy access  
- ✅ **Hero Section** – Includes a heading, paragraph, and button  
- ✅ **Cards Section** – Displays information in a structured layout  
- ✅ **Featured Work Section** – Highlights important projects with images and descriptions  
- ✅ **Footer with Social Links** – Contains social media icons and copyright info
